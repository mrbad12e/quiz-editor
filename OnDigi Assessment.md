# Quiz Editor

Welcome to the OnDigi interview!
This assessment is the first stage of the interview process.
As it is important for us to know your practical skills,
please take your time and try your best to make it as good as you can.

## Description

The quiz editor is a tool that allows creating a new quiz or updating an existing one.
The output is a JSON file containing the quiz data.
The output file can be imported as the input to allow updating the quiz.

A quiz should have:
 - Name
 - Description
 - A list of multiple-choice questions

Each multiple-choice question should have:
 - Name
 - Description
 - Sort order
 - A list of options (at least 2 options)
 - A list of values of correct options (at least 1 option value)

Each option should have:
 - Value
 - Label
 - Sort order
 
## Acceptance Criteria

- The quiz editor should allow users to create and update quizzes as described
- UI: It doesn't need to be fancy, but should be user-friendly
- UX: It should be intuitive, easy to use for end users
- It should be created using React library
- UI libraries (MUI, Ant, Shad,...) are not allowed
